# TOEIC Setup Instructions

## Complete Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/esbisb-source/toeic.git
   cd toeic
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```

## Configuration Guide
- Configuration files can be found in the `config` directory.
- Update `config.json` with your API keys and other relevant settings.
- Ensure the database connection settings are correct in the `db_config.json` file.

## Usage Documentation
- To use the application, follow the instructions provided in the application UI.
- You can find additional command-line options by running:
   ```bash
   npm run help
   ```

For detailed features and functionalities, refer to the documentation within the app.
